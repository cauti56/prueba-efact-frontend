# Proyecto EFACT - Visualizador de Documentos

Visualizador de Documentos – Prueba Técnica EFACT

Este proyecto es la solución para la prueba técnica de EFACT, donde se implementa un visualizador de documentos electrónicos (PDF, XML, CDR) conectándose al backend de EFACT mediante un servidor proxy local.

#Tecnologías Usadas

Frontend
Angular: 21.0.2
Typescript: Última versión incluida en Angular 21
RxJS: Incluido en Angular
HTML5 / CSS3
Angular Standalone Components

Backend
NodeJS: 22.18.0
ExpressJS
Axios
CORS
Servidor liviano actuando como proxy hacia EFACT para evitar CORS
